🔐 Google Refresh Token 획득

⚠️ 주의: 이 페이지는 선생님만 사용해야 합니다.
Refresh Token은 서버에 한 번만 설정하면 학생들은 로그인 없이 사용할 수 있습니다.

Step 1: Google OAuth Playground 접속

아래 링크를 클릭하여 Google OAuth 2.0 Playground에 접속하세요:

Step 2: 설정

  1. 오른쪽 상단 톱니바퀴(⚙️) 클릭
  2. "Use your own OAuth credentials" 체크
  3. 입력할 정보:
    OAuth Client ID: 129459484885-49jhhorvjq9cbd1nhjnf4qlrslqchdj7.apps.googleusercontent.com OAuth Client secret: (Google Cloud Console에서 확인)
⚠️ 중요: redirect_uri_mismatch 에러가 나면:
Google Cloud Console → Credentials → OAuth 2.0 Client ID → Authorized redirect URIs에
https://developers.google.com/oauthplayground 추가

Step 3: 권한 선택

  1. 왼쪽 목록에서 "Drive API v3" 찾기
  2. https://www.googleapis.com/auth/drive.readonly 선택
  3. "Authorize APIs" 버튼 클릭
  4. Google 계정으로 로그인 (shin.jeiths@gmail.com)

Step 4: Refresh Token 획득

  1. 인증 완료 후 "Exchange authorization code for tokens" 클릭
  2. 오른쪽에 표시된 Refresh token 복사
  3. 아래 형식으로 저장:
    Refresh token: 1//0xxxxxxxxxxxxx...

Step 5: Vercel 환경변수 설정

획득한 토큰들을 Vercel에 설정합니다:

# Vercel Dashboard → Settings → Environment Variables GOOGLE_CLIENT_ID=129459484885-49jhhorvjq9cbd1nhjnf4qlrslqchdj7.apps.googleusercontent.com GOOGLE_CLIENT_SECRET=GOCSPX-xxxxxxxxxxxxxxxx GOOGLE_REFRESH_TOKEN=1//0xxxxxxxxxxxxx... GOOGLE_DRIVE_FOLDER_ID=1rEMeET9wqGR2Ky-fefFm6BumbXsRBi77

또는 Vercel CLI 사용:

vercel env add GOOGLE_REFRESH_TOKEN

✅ 설정 완료 후

  • 서버가 자동으로 Google Drive에 접근
  • 학생들은 로그인 없이 프로젝트 사용 가능
  • 모든 API 키는 서버에서만 처리되어 안전
